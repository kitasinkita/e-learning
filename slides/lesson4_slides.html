📦 Playwrightのインストール

1️⃣プロジェクトフォルダ作成

mkdir playwright-project
cd playwright-project

2️⃣Playwrightインストール

npm init -y
npm install playwright

3️⃣ブラウザのダウンロード

npx playwright install
⚠️ ブラウザのダウンロードには時間がかかります（約300MB）
✅ これで準備完了！Claude Codeから使えるようになりました

🚀 基本的な使い方

📝基本コード構造

const { chromium } = require('playwright'); (async () => { // ブラウザを起動 const browser = await chromium.launch(); // 新しいページを開く const page = await browser.newPage(); // サイトにアクセス await page.goto('https://example.com'); // スクリーンショット撮影 await page.screenshot({ path: 'example.png' }); // ブラウザを閉じる await browser.close(); })();

主要な操作メソッド

  • page.goto(url) - URLにアクセス
  • page.click(selector) - 要素をクリック
  • page.fill(selector, text) - テキスト入力
  • page.screenshot() - スクリーンショット

🔧 よくあるトラブル＆解決法

❌ エラー：タイムアウト
  • ✅ waitForSelector()で要素の表示を待つ
  • ✅ タイムアウト時間を延長（timeout: 30000）
  • ✅ ネットワーク速度を確認
❌ エラー：要素が見つからない
  • ✅ セレクタの記述を確認
  • ✅ ページが完全に読み込まれるまで待つ
  • ✅ iframeの中の要素か確認
❌ エラー：ブラウザが起動しない
  • ✅ npx playwright install を再実行
  • ✅ headless: false でデバッグ
  • ✅ アンチウイルスソフトの設定確認
💡 デバッグのコツ：{ headless: false } でブラウザを表示させて動作確認
