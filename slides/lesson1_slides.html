📥 詳細インストール手順

1️⃣Visual Studio Code

✅ ブラウザで https://code.visualstudio.com にアクセス
✅ 「Download for Windows」をクリック
✅ ダウンロードしたファイル（VSCodeUserSetup-x64-1.x.x.exe）をダブルクリック
✅ ライセンス同意 → 「次へ」→「次へ」→「インストール」

2️⃣Git Bash

✅ ブラウザで https://git-scm.com にアクセス
✅ 「Download for Windows」をクリック
✅ ダウンロードしたファイル（Git-2.x.x-64-bit.exe）を実行
✅ 全て「Next」で進んでOK（デフォルト設定で大丈夫）
💡 各インストール完了後、PC再起動を推奨
📚 より詳しい環境構築を実施したい場合
IT OPTIMIZATION所属の渡辺が提供するGitHubリポジトリを実行しても構いません。
https://github.com/Ryoji-Watanabe-itopt/claude-code-dev-environment

📗 Node.js詳細インストール

3️⃣Node.js（NPM付き）のインストール

✅ ブラウザで https://nodejs.org にアクセス
✅ 「LTS」版（長期サポート版）をクリックしてダウンロード
✅ ダウンロードしたファイル（node-v20.x.x-x64.msi）をダブルクリック
✅ Welcome画面で「Next」
✅ ライセンス同意にチェックして「Next」
✅ インストール場所はデフォルトのまま「Next」
✅ 機能選択もデフォルトのまま「Next」
✅ 「Install」をクリック
💡 NPMとは？
Node.jsに付属するパッケージ管理ツールです。様々なプログラミングツールやライブラリを簡単にインストールできます。

📋 インストール確認

✅ Windowsキー + R → 「cmd」と入力してEnter
✅ コマンドプロンプトで以下を実行:
node --version
npm --version
✅ 両方でバージョン番号が表示されればインストール成功！

💻 Claude CodeをVS Codeで使う

4️⃣VS Code内でClaude Codeをインストール

✅ Visual Studio Codeを起動
✅ VS Codeでターミナルを開く：Ctrl + `（バッククォート）
✅ または メニュー → 「表示」→「ターミナル」

5️⃣ターミナルでClaude Codeをインストール

✅ VS Code下部のターミナル画面で以下のコマンドを実行:
npm install -g claude-code
✅ インストール完了まで待つ（数分かかる場合があります）

6️⃣Claude Codeを起動

✅ 同じターミナルで以下のコマンドを実行:
claude
✅ 初回起動時はAPI設定画面が表示される場合があります
✅ 画面の指示に従って設定を完了してください
🎉 これでVS Code内のターミナルでClaude Codeが使えるようになりました！
コードエディタとAIアシスタントを同じ画面で使える便利な環境の完成です！

🎯 VS Codeで実習：Hello Worldを作ろう！

💻VS Codeでの作業の流れ

✅ VS Codeで新しいファイルを作成（Ctrl + N）
✅ 「.js」拡張子で保存（例：hello.js）
✅ VS Code内のターミナルでClaude Codeを起動（`claude`コマンド）
✅ Claude CodeでAIにプログラムを作成してもらう
✅ 生成されたコードをエディタにコピー＆ペースト
✅ ターミナルで `node hello.js` でプログラム実行

📝課題1：あいさつプログラム

VS Code内のターミナルでClaude Codeに話しかけてみよう：
「JavaScriptで、ユーザーの名前を聞いてあいさつするプログラムを作成してください」

🧮課題2：計算プログラム

次にClaude Codeに話しかけてみよう：
「JavaScriptで、2つの数字を足し算する電卓プログラムを作成してください」
💡 VS Codeの便利な機能
• エディタとターミナルを同時に表示
• コードを書いてすぐに `node ファイル名.js` で実行
• エラーがあればClaude Codeが修正を提案
• Ctrl + ` でターミナルの表示・非表示を切り替え
