🎯 実践プロジェクト①

📰
ニュース自動収集ツール
複数のニュースサイトから最新情報を収集し、
カテゴリ別に整理してCSVファイルに保存。
毎日自動実行して、GitHubにアップロード。
Claude Code Playwright MCP GitHub

機能要件

  • ✅ 3つ以上のニュースサイトを巡回
  • ✅ タイトル、URL、日時、カテゴリを取得
  • ✅ CSV形式で保存（日付別）
  • ✅ 重複記事の除外
  • ✅ GitHubに自動アップロード

💻 プロジェクト①の実装

📝Claude Codeへの指示

「以下の機能を持つニュース収集ツールを作って： 1. Yahoo、Google、日経のニュースサイトを巡回 2. 各サイトから最新10件のニュースを取得 3. タイトル、URL、取得日時、カテゴリをCSVに保存 4. 重複したニュースは除外 5. news_YYYY-MM-DD.csvという名前で保存 6. 実行後、自動でGitHubにコミット＆プッシュ Playwrightを使って実装して、 エラーハンドリングも入れて」

期待される出力

📁 news_2024-01-15.csv 
タイトル,URL,カテゴリ,取得日時
新技術発表で株価上昇,https://...,経済,2024-01-15 10:30
スポーツ大会開催決定,https://...,スポーツ,2024-01-15 10:31

🎯 実践プロジェクト②

📝
Webフォーム自動化ツール
定型的なWebフォームへの入力を自動化。
複数のパターンを登録可能で、
入力後のスクリーンショットも自動保存。
Claude Code Playwright JSON設定 画像処理

機能要件

  • ✅ JSONファイルから入力データを読み込み
  • ✅ 複数のフォームパターンに対応
  • ✅ 入力前後でスクリーンショット撮影
  • ✅ エラー時の自動リトライ
  • ✅ 実行ログの記録

💻 プロジェクト②の実装

⚙️設定ファイル（form_data.json）

{ "forms": [ { "name": "お問い合わせフォーム", "url": "https://example.com/contact", "fields": { "name": "山田太郎", "email": "yamada@example.com", "message": "テスト送信です" } } ] }

実装のポイント

JSONファイルから設定を読み込む
各フォームに対してPlaywrightで自動入力
スクリーンショットを before/after で保存
実行結果をログファイルに記録

🔐 セキュリティ・注意事項

❌ 絶対にやってはいけないこと
  • パスワードをコードに直接書く
  • APIキーをGitHubに公開
  • 他人のサイトに負荷をかける
  • 利用規約に違反する自動化
✅ 推奨される対策
  • 環境変数（.env）を使用
  • .gitignoreで機密情報を除外
  • 適切な待機時間を設定
  • robots.txtを確認
